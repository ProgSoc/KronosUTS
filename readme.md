# Kronos - A ProgSoc Project

The result of frustration with UTS' existing timetable viewer, a backend for retrieving and storing data and a frontend to visualize it.

Inspired by Notangles by USNW csesoc.

## Stack

### Backend

- NestJS
- Prisma
- Fastify

### Frontend

- React
- Vite
- React Query
- React Router
